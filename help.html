Creating Your Project

When your client is happy with the quote you gave them, and they give you the go ahead, we can begin the automated process of getting the domain / hosting services setup for the website you plan on making.



Click the  Create New Project button

Fill in all required Information

  1. Project ID  - This is the Identification number for your project whenever a customer wants to reference to it. It’s recommended that you do not change this as it will be automatically generated for you

  2. Designer Email Address -  This HAS to be an employee of Jaxify Software, this email is your email that was used to sign into this system. If you won’t be working on this project but rather one of your colleges, you would enter their email. This is the email the project will be assigned to. So if you’re going to make this website, you enter your own email address.

  3. Client Full Name -  Enter the name of the client. If this information has not been provided to you, it’s recommended that you enter their business name. For example Jaxify Software, JBL Plumbers, John’s Electricals, etc

  4. Client Email Address -  This field is required as it’s how the system will communicate any and all processes to the client. As you work through the project, the client will receive communication emails on exactly where their website progress is at.

  5. Client Invoiced -  This is a numeric field where you have to specify which estimated value the client agreed upon, so if they agreed to do the project for R1500 , you would enter the value ‘1500 ’ without the “R ”

  6. Invoice Instructions -  This is one of the most important fields. You need to enter exactly what you’re doing for the client, and how much you’ve charged them.

    1. So if the client wants 3 emails, a 1 page website, hosting, 1 video and maintenance, you’d fill in the following.

      1. 3 Emails (R200)/Annually (or whatever you charged them)

        1. info@example.co.za

        2. james@example.co.za

        3. bob@example.co.za

      2. Hosting (R400)/ Annually (or whatever you charged them)

      3. 1 Promo Video (R350) 

    2. This is so we can send the client a formal invoice and know exactly what to set up in the backend.

    3. Once you receive the proof of payment. Forward it on to Jaco (0828826870) Mark the project as paid via your dashboard, or wait for this to happen automatically when the EFT is processed. You are allowed to accept cash, however, it’s utterly important that you deposit the full invoiced amount into the appropriate bank account.

  7. Due Date -  This is a calendar field where you can fill in an estimate on when you think you will be done. It’s better to set this in a future date and complete it sooner than a date you won’t be able to complete it by.

  8. Reserved Domain Name -  This is the most important part. You will ask the client for a domain name (Web Address) that they want for their website. Our company pays for this fee. You can check available domains via www.godaddy.co.za  

    1. It’s essential that you spell the domain correctly when entering it here.

    2. A domain NEEDS to be in this format google.com / jaxifysoftware.com / yahoo.com / website.co.za / udemy.com - There should be no ‘/’ nor www nor https.

    3. If you set this up incorrectly, a chunk of what’s due to you will be taken from your payout to buy a new domain with the correct details, thus it’s essential that you make sure you enter the correct domain.

  9. Add Project -  Finally, you can add your project and it will be loaded onto the system.


Awaiting Approval

Now that you’ve added your project. The project and all its information will be sent to a manager, which will be approved anywhere between 5 minutes and 12 hours.


Once approved, you’ll get an email notifying you and the client will receive their invoice.


We require all clients to make the payment first, before we can start setting anything up on our side. Since you’re able to talk to the client directly, we expect you to mark the project as paid on your dashboard by clicking the green button under  Update Status. You should only do this when they send you a proof of payment screenshot / pdf.


Should they not do this, we have an automated system which will automatically mark it as paid once we receive the EFT Payment.


Once the project is marked as paid. The client will receive an update informing them.


Requesting Domain Purchase

After payment is complete, it’s expected that you double check what domain the client wants. You’ll be able to click the green button under  Update Status to do this. You will be asked to confirm the domain you want purchased. This will automatically begin the setup process for purchasing the domain, assigning the domain to our hosting platform and creating the wordpress website. This process takes anywhere between  1 hour and  24 Hours. Both you and the client will be informed when this step is complete.


The project will now be marked as “In Progress” and you’ll be sent a link to edit the website with login information to your email. You’ll also be able to edit the website from your employee dashboard by clicking “ Edit Website” 


Working On The Project

  • You’ll be making use of the WordPress Platform and use the Brizy Pro Page Builder. A Tutorial can be found here. -  https://youtu.be/IPZR_vWqc7s  

Completing the Project

Once the project is completed, it’s expected that you add their email as a user with a random password under the role:  Administrator

This is to give them full access to the website Administrative Dashboard.

Once everything is completed, you’ll navigate to your employee dashboard and mark the project as completed. Which will set the project to be paid out to you by the 25th of the specific month. Payments may take up to 48 Hours to process.


The client will receive an email giving them their login information automatically.
